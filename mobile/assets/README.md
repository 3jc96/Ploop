# Mobile App Assets

Place the following assets in this directory:

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen (1242x2436)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

Startup sound:
- The app plays a short synthesized "ploop" sound on launch (generated at runtime).

For now, you can use placeholder images or generate them using:
- [App Icon Generator](https://www.appicon.co/)
- [Expo Asset Generator](https://docs.expo.dev/guides/app-icons/)
